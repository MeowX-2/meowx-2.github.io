VimTeX

Basically LaTex setup with nvim with zathura as the pdf viewer. Zathura is minimal, more importantly it supports SyncTeX: You can click a certain line in the pdf and that will take you to the line corresponding to the .tex file.

Note— It’s also important to install both zathura and zathura-pdf-poppler plugin.

In ~/.config/zathura/zathurarc add the following– 

set synctex true
set synctex-editor-command "nvr --remote-silent +%{line} %{input}"

The code below sets up the default pdf viewer for VimTeX in NeoVim. Add these in ~/.config/nvim/init.lua– 

vim.g.vimtex_view_method = 'zathura'
vim.g.latex_view_general_viewer = 'zathura'
vim.g.vimtex_compiler_progname = 'nvr'
